# Foretell v2.1 - Clean Deployment

## Files Included:
- api/kalshi-markets.js - Backend API
- index.html - Entry point  
- package.json - Dependencies
- vercel.json - Vercel config

## Deploy Instructions:

### Option 1: GitHub Upload
1. Delete everything in your GitHub repo EXCEPT .git folder
2. Upload these 4 files/folders to GitHub root
3. Commit
4. Vercel auto-deploys

### Option 2: Vercel CLI
1. cd to this folder
2. vercel login
3. vercel --prod
